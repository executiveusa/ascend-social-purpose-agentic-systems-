# Flywheel Overlay

This repo is designed to be maintained on an Agentic Coding Flywheel Setup VPS.

ACFS provides the developer environment. Asc3nd Mission OS provides the nonprofit product.

Use this separation:

- `/data/projects/asc3nd-social-purpose-os` — source repo.
- `mission-data` volume — runtime tenant data.
- `icm` volume — tenant workspaces.
- `.beads` — product tasks.
