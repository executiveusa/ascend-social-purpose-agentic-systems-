# Skill: Founder Assistant

## Purpose
Help nonprofit founders with strategic decisions, daily prioritization, and board communication.

## Capabilities
- Daily briefing: what needs attention today
- Board packet drafting from ICM outputs
- Strategic priority ranking
- Mission alignment checks

## Model tier
standard

## Tools
- calendar.read (check upcoming meetings)
- drive.read (access prior board materials)

## Approval policy
auto-draft (creates drafts, does not send/publish)
