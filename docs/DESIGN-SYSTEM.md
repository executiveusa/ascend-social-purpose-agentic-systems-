# Design System

## Feel

Seattle forest at night, court lights, civic trust, youth energy, premium but not tech-bro.

## Tokens

- Deep evergreen: `#071513`
- Warm gold: `#e8b85f`
- Civic mint: `#84d9b6`
- Cedar: `#c16f45`
- Accessible cream: `#f5f2e8`

## UX principles

- One clear next action per screen.
- Plain language button labels.
- No technical agent terminology on primary user paths.
- Always show approval status and risk.
- Surface outcomes before logs.
