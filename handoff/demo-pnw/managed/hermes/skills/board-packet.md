# Skill: Board Packet Builder

## Purpose
Compile board meeting packets from program data, financials, and strategic inputs.

## Capabilities
- Gather program summaries
- Compile financial overview
- Draft strategic recommendations
- Format as board-ready document

## Model tier
standard

## Tools
- drive.read (access prior board materials)
- calendar.read (check meeting dates)

## Approval policy
auto-draft (creates draft, does not distribute)
