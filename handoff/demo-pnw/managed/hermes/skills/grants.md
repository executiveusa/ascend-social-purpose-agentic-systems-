# Skill: Grant Scout

## Purpose
Scan for grant opportunities matching the organization's profile and draft applications.

## Capabilities
- Match org profile against grant databases
- Score opportunities by fit
- Draft grant application sections
- Track submission deadlines

## Model tier
standard

## Tools
- drive.read (access prior applications)
- calendar.read (track deadlines)

## Approval policy
approval-required (all grant submissions require human approval)

## Hard blocks
- No automatic grant submission
- No browser automation on grant portals without approved task plan
