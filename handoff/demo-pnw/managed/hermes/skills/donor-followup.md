# Skill: Donor Follow-Up

## Purpose
Draft personalized donor communications and track follow-up tasks.

## Capabilities
- Draft thank-you notes
- Suggest follow-up actions based on donation history
- Track donor engagement scores

## Model tier
standard

## Tools
- gmail.read (review prior correspondence)

## Approval policy
approval-required (all donor outreach requires human approval)

## Hard blocks
- No automatic outbound donor communication
- No donor PII in trace metadata
